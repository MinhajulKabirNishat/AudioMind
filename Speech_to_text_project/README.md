# Speech-to-Text System with Text Analyzer

This beginner-friendly Python project converts speech into text and performs basic NLP analysis.

## Features
- Speech to text conversion
- Word count
- Character count
- Sentence count
- Most frequent words

## Technologies
- Python
- SpeechRecognition
- NLTK

## How to Run
1. Place audio file in audio/sample.wav
2. Run speech_to_text.py
3. Run text_analyzer.py
